# Conventional Agent for Mental Health Triage

## 📌 Project Description
This project implements a Conventional Mental Health Triage Agent using Python and Flask.  
The system asks mental health assessment questions, calculates a score, classifies the risk level, 
and provides recommendations based on user responses.

The project demonstrates rule-based Artificial Intelligence, decision making, 
and web-based interface development.

---

## 🎯 Objectives
- To design a rule-based conversational agent
- To perform mental health assessment using questionnaire
- To classify severity level (Low, Moderate, High, Emergency)
- To provide recommendations based on score
- To create a web interface using Flask

---

## 🧠 Features
- Questionnaire-based assessment
- Score calculation
- Risk classification
- Severity interpretation
- Recommendations generation
- Result logging in CSV file
- Web-based interface

---

## 🛠 Technologies Used
- Python
- Flask
- HTML (template string)
- CSV file handling
- Rule-based AI logic
